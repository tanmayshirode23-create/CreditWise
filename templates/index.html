About CreditWise

CreditWise is an intelligent loan approval prediction system that leverages machine learning to assess loan applications. Our system analyzes multiple factors including income, credit score, employment status, and financial history to provide accurate predictions.

The model uses Logistic Regression with feature engineering to achieve an impressive 88% accuracy in predicting loan approvals. This helps financial institutions make faster, more reliable lending decisions.
